Приложение для идентификации лиц
Автор: Денисов Вадим Алексеевич
Научный руководитель: Воронин Игорь Вадимович

Как запускать:
1) Установить зависимости: выполнить pip install -r requirements.txt (из папки с приложением)
2) Положить изображения, которые будут использоваться для обучения модели в папку ./data/to_label, а изображения для распознавания лиц на них в папку ./data/to_recognize
3) Запустить приложение: выполнить python3 main.py (из папки с приложением)
4) Далее приложение спросит пользователя, что он хочет сделать:

    -Загрузить новые изображения в базу данных (нужно ввести L)
    -Распознать лица на изображениях (нужно ввести R)

   При выборе первого варианта приложение спросит, каким образом пользователь хочет указать классы для изображений (имена людей, которым принадлежат лица).
   Пока что реализован только вариант с указанием классов вручную (нужно ввести M): приложение начнёт итерироваться по всем изображениям в папке to_label. Для каждого изображения приложение выведет количество лиц, найденных на нём, а затем попросит по одному ввести классы, которым принадлежат распознанные лица СЛЕВА НАПРАВО (!)
   
   При выборе второго варианта приложение начнёт распознавание лиц на изображениях из папки to_recognize
5) Для завершения работы с приложением нужно ввести F, когда такая опция будет доступна
6) Результаты работы приложения будут лежать в папке ./data/result:
    
    -В папке images_with_boxes оригинальные изображения с распознанными лицами и подписанными классами
    -В папке csv_with_labels csv-файл с двумя столбцами: название изображения и лица, распознанные на нём (слева направо)
